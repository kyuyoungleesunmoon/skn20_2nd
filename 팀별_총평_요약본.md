# 팀별 AI 프로젝트 총평 요약

## 평가 개요
- **평가 대상**: SKN20 2기 AI 프로젝트 각 팀
- **평가 기준**: 데이터 전처리(30점) + 모델 학습(40점) + 모델 평가(30점) = 총 100점
- **평가 방식**: 소스코드, 산출물, 문서 종합 분석

## 2팀 총평 (500자 내외)

**프로젝트**: 헬스장 회원 이탈 예측  
**최종 점수**: 96/100점

2팀은 헬스장 회원 이탈 예측 프로젝트를 통해 AI 프로젝트의 전 과정을 매우 우수하게 수행했습니다. 

**데이터 전처리(29/30)**: 4,002개 샘플에 대한 체계적인 EDA로 데이터 특성을 완벽히 파악했습니다. 클래스 불균형(2.33:1)을 SMOTE로 해결하고, StandardScaler로 정규화했으며, 11개의 도메인 기반 파생 특성(Lifetime_per_Month, Class_Engagement 등)을 생성하여 모델 성능 향상의 기반을 마련했습니다.

**모델 학습(38/40)**: 6개 ML 모델과 1개 DL 모델을 비교하고, RandomizedSearchCV로 XGBoost(+33.1%)와 LightGBM(+34.2%)을 최적화했습니다. Stacking Ensemble을 구축하여 F1 Score 0.9657, AUC-ROC 0.9712를 달성했습니다.

**모델 평가(29/30)**: 다양한 지표로 모델을 다각도 분석하고, 특성 중요도(Lifetime 18.47%, Month_to_end_contract 15.23%)를 통해 비즈니스 인사이트를 도출했습니다. Streamlit 대시보드를 구현하여 실무 적용 가능성을 입증했으며, ROI 분석($107,000 절감)까지 제시했습니다.

전체적으로 데이터 분석부터 모델링, 평가, 배포까지 전 과정이 매우 체계적이고 전문적이며, 실무 적용 가능성이 높은 우수한 프로젝트입니다.
